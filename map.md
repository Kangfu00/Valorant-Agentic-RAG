# คู่มือแผนที่ VALORANT ทั้ง 12 แผนที่

## บทสรุปผู้บริหาร

Riot ปัจจุบันแสดงแผนที่มาตรฐานแบบ Spike ทั้งหมด 12 แผนที่บนหน้า Maps ได้แก่ Corrode, Abyss, Sunset, Lotus, Pearl, Fracture, Breeze, Icebox, Ascent, Split, Haven และ Bind; Corrode ถูกเพิ่มเข้ามาในแพตช์ 11.00 ในฐานะแผนที่ใหม่ล่าสุดของโหมดมาตรฐาน ณ เวลานั้น ส่วน **ranked pool ปัจจุบันในแพตช์ 12.08** ใช้ 7 แผนที่คือ **Ascent, Breeze, Fracture, Haven, Lotus, Pearl และ Split** โดย Riot ยืนยันว่าแพตช์ 12.08 สลับ **Ascent เข้า / Bind ออก** จากคิว Competitive และ Deathmatch. citeturn5view0turn10view0turn14search16turn1view0

ถ้ามองเชิงโครงสร้าง แผนที่ของ VALORANT แบ่งได้เป็น 4 กลุ่มใหญ่:  
แผนที่ที่ “ต้องชนะ mid” เช่น Ascent, Split, Sunset, Pearl; แผนที่เปิดโล่งยิงไกลอย่าง Breeze และ Abyss; แผนที่ที่มีกลไกเฉพาะชัดเจนอย่าง Bind, Lotus, Icebox และ Fracture; และแผนที่สามไซต์อย่าง Haven และ Lotus ที่เน้นการดึงจังหวะหมุนของฝั่งรับให้เสียสมดุล. citeturn5view0turn6view0turn8view0turn9view0

ข้อผิดพลาดที่ทำให้แพ้รอบมากที่สุดไม่ใช่ “aim ไม่คม” แต่คือ **จังหวะ execute ผิด**: บุกโดยไม่ยึด choke สำคัญ, plant โดยไม่เคลียร์ lane retake หลัก, หรือใช้ utility หมดก่อนตอนเข้าพื้นที่จริง สำหรับ ranked ที่อยากจำให้ใช้ได้จริงที่สุด ให้จำเพียง **1 default + 1 fast hit + 1 retake pattern ต่อแผนที่** ก็เพียงพอแล้วสำหรับการไต่แรงค์อย่างมีระบบ

## คู่มือรายแผนที่

### Ascent

- **ภาพรวม:** Ascent เป็นแผนที่ 2 ไซต์แบบสามเลนที่เน้นสงครามยึดพื้นที่ตรงกลางแผนที่อย่างมาก และมีประตูไซต์ที่ปิดได้แบบ “พังแล้วพังเลย” ทำให้การคุม mid และการแยกบุก (split) สำคัญกว่าการ 5-man ชน choke ตรงๆ; Red Bull ยังจัด Ascent เป็นหนึ่งในแผนที่ที่ค่อนข้างเอียงไปทางฝั่งรับ. citeturn5view0turn9view0
- **เกมบุก:** execute ที่เสถียรที่สุดคือ **A split (A Main + Cat/Tree)** หรือ **B split (B Main + Mid/Market)** มากกว่า “ยัด main ตรงๆ” เสมอ; default ที่ดีคือให้ initiator probe Mid Top/Tiles, controller เตรียม smoke Market/CT หรือ Tree/Heaven และ duelist เป็นคนเปิดมุมแรกหลัง utility ลงครบ จุดที่ควรยึดให้ได้คือ **Top Mid, Pizza/Cat, Market, Tree, Switch**.
- **เกมรับ:** setup มาตรฐานคือ **2-1-2** หรือ **2A / 1 Mid / 2B**; ฝั่งรับที่ดีต้องมีผู้เล่นคุม Mid ตลอด เพราะถ้าเสีย Mid จะโดน split ได้ทั้งสองไซต์ Anchor ที่มีมูลค่าสูงคือ **A Generator/Dice/Tree** และ **B Switch/Boat/Back Site**; retake ให้เล่นคู่จาก **Heaven+Tree** หรือ **Lane+Market** แทนการเดี่ยวทีละมุม.
- **Meta ตอนนี้:** Ascent กลับเข้าสู่ pool Ranked ในแพตช์ 12.08 และ Riot เคยแก้ผนัง **B Main** ให้ยิงทะลุไม่ได้เพื่อลด barrier-drop spam ซึ่งทำให้ฝั่งบุกมีพื้นที่เริ่มต้นปลอดภัยขึ้นเล็กน้อย แผนที่นี้จึงยังเหมาะมากกับ controller แบบ Omen/Brim, initiator ที่คุม mid ได้ และ sentinel ที่ล็อกไซต์ปลายทางได้ดี. citeturn1view0turn13search8
- **Do / Don’t:** **Do:** เล่นเพื่อยึด Mid ก่อนค่อยตัดสินใจ hit. **Don’t:** 5 คนวิ่งชน A Main หรือ B Main แบบไม่มี split และไม่มีคนกัน flank.
- **3 Pro Tips:** 1) ถ้าชนะ Mid แล้ว B execute จะเสถียรกว่าการเดิน B Main ตรงๆ มาก, 2) อย่าปิดประตู Tree/Market แบบอัตโนมัติทุกครั้ง—ปิดเมื่อมัน “ตัด fight” ให้คุณจริงๆ, 3) หลัง plant ฝั่งรับจะ retake เป็นแพทเทิร์นชัดมาก ให้เผื่อ utility สำหรับ Lane/Heaven เสมอ

**TL;DR:** Ascent ชนะด้วย **mid control + split timing**; ทีมที่ยัด main ตรงๆ โดยไม่คุม mid มักทำรอบให้ยากขึ้นเอง

### Breeze

- **ภาพรวม:** Breeze เป็นแผนที่ 2 ไซต์ขนาดใหญ่ พื้นที่เปิดเยอะ ระยะยิงไกลมาก และมี mid ใหญ่ที่มีอิทธิพลต่อทั้งสองไซต์; Riot ระบุใน 12.00 ว่าแผนที่นี้ถูก rework เพื่อลดความซับซ้อนของมุม เพิ่มความกระชับของ open space และช่วยให้ฝั่งรับหมุนได้ดีขึ้น แต่แกนหลักเรื่อง “open map, long sightlines” ยังอยู่ครบ. citeturn5view0turn9view0turn10view0
- **เกมบุก:** โครงบุกที่ดีที่สุดคือ **1-3-1** หรือ default ที่ให้คนคุม **Mid Doors/Nest** และกดหนึ่ง extremity เงียบๆ ก่อน จากนั้นค่อยเลือก **A ผ่าน Cave + Halls split** หรือ **B ผ่าน Main + Mid Top** controller แบบ wall มีค่ามากที่สุด เพราะช่วยตัด line of sight จำนวนมากในจังหวะเดียว จุดที่ควรยึดให้ได้คือ **Nest, Mid Doors, Mid Top, A Halls, B Tunnel**.
- **เกมรับ:** ผู้เล่น mid สำคัญมาก ถ้าไม่มีคนยืน/เช็ค **Nest-Doors** ฝั่งบุกจะผ่ากลางแล้วเปิดทั้งสองไซต์พร้อมกันได้ง่าย Anchor ที่ดีควรเล่น contact + fallback มากกว่ายืนลึกโดดเดี่ยว และการ rotate ควรเริ่มเร็วกว่าแผนที่เล็กๆ เพราะถึงจะ rework แล้ว แต่ระยะ still กว้างกว่าค่าเฉลี่ยของเกมอยู่ดี.
- **Meta ตอนนี้:** Breeze อยู่ใน pool ปัจจุบัน และหลัง rework ใน 12.00 มันยังเป็นบ้านของ **Viper/Harbor, Sova/KAYO, Jett** หรือใครก็ตามที่ชนะ duel ระยะไกลได้ ข้อผิดพลาดคลาสสิกคือเล่นช้าเกินไปในพื้นที่เปิด หรือแยกตัว lurk ลึกโดยไม่มี utility ช่วยข้าม lane เปิด. citeturn10view0turn14search16
- **Do / Don’t:** **Do:** เอา mid information ให้ได้ทุก rifle round. **Don’t:** ดรายพีคข้ามพื้นที่เปิดกว้างโดยไม่มี smoke/wall หรือ recon นำ.
- **3 Pro Tips:** 1) ถ้า defense ไม่ใส่แรงกดที่ Nest ให้รีบกิน Mid Doors เร็ว, 2) B hit ที่ดีมักแข็งแรงขึ้นมากเมื่อมาจาก **Mid Top + B Main** พร้อมกัน, 3) บน A ถ้าคุณไม่คุม Halls ได้ post-plant จะเปราะกว่าที่คิด

**TL;DR:** Breeze ไม่ได้ชนะด้วยการ “ไถยาว” แต่ชนะด้วย **mid ownership, wall timing และการคุมระยะยิง**

### Fracture

- **ภาพรวม:** Fracture เป็นแผนที่ 2 ไซต์แบบ asymmetric ที่ฝ่ายบุกเริ่มเข้าล้อมจากสองด้านของแผนที่ได้ ทำให้มันไม่ใช่ “เดินเข้าหน้าบ้าน” แบบ map ปกติ แต่เป็น map ที่บังคับให้ทั้งสองทีมคิดเรื่อง pinch, spawn-cut และการสื่อสารมากเป็นพิเศษ. citeturn5view0turn9view0
- **เกมบุก:** route ที่ชัดที่สุดคือ **A Main + Dish** หรือ **B Main + Arcade** เพื่อบีบ anchor ให้ต้องหันคนละทิศ execute ที่ดีต้องมี smoke ปิด **Spawn/Canteen/Tower** ตามไซต์ และมี initiator utility ล้าง close corners ก่อน duelist หลุดเข้าไป จุดที่ควรเอาให้ได้คือ **Dish, A Drop, Arcade, Sands, B Canteen**; map นี้ reward ทีมที่ “เข้าพร้อมกัน” มากกว่าทีมที่ค่อยๆ ไหล.
- **เกมรับ:** setup ที่นิยมคือ **2-2-1** หรือมีหนึ่งคน flex ข้อมูลกลางแผนที่ ฝั่งรับที่นิ่งเกินไปจะโดน pinch ง่าย จึงควรมี early info play ใน **A Main** หรือ **B Arcade** ด้วย utility ที่เทรดกันได้ Anchor ไม่ควรตายบน first angle ถ้าโดน utility dump ให้ถอยไปเล่น Tower/Canteen/retake layer.
- **Meta ตอนนี้:** Riot นำ Fracture กลับเข้าคิวใน 12.05 และมันยังเป็นแผนที่ของ **Breach** มากที่สุด map หนึ่งในเกม เพราะ stun/flash line ผ่านกำแพงเปิดไซต์ได้แรงมาก ส่วน Neon/Raze และ controller ที่ปิด spawn route เก่งก็มีมูลค่าสูงมากเช่นกัน. citeturn2view0turn14search16
- **Do / Don’t:** **Do:** มองไซต์เป็น “โจทย์การบีบสองด้าน” ไม่ใช่ทางเข้าเดี่ยว. **Don’t:** ให้คนเข้าเร็วก่อน utility หรือแยกชิ้นส่วนของ split จนเทรดกันไม่ได้.
- **3 Pro Tips:** 1) เก็บ flash/stun ไว้อย่างน้อยหนึ่งจังหวะสำหรับ post-plant จะคุ้มกว่าโยนหมดหน้าไซต์, 2) defense ควร push/contest เป็นคู่ ไม่ใช่ solo gamble, 3) ถ้าคุณทำ noise ฝั่งหนึ่งได้ อีกฝั่งของแผนที่จะเปิดให้ pinch ง่ายมาก

**TL;DR:** Fracture คือแผนที่ของ **timed split hit**; ถ้าเข้าพร้อมกันมันจะง่ายมาก แต่ถ้าไม่พร้อมกันมันจะพังทันที

### Haven

- **ภาพรวม:** Haven เป็นหนึ่งในสองแผนที่สามไซต์ของเกม และ Riot อธิบายชัดว่า defenders สามารถใช้พื้นที่ที่เพิ่มขึ้นสำหรับการ push เชิงรุกได้ ขณะที่ Red Bull ชี้ว่า defenders มักถูกดึงบางเกินไป และ attackers จะได้ประโยชน์มากจากการ fake และการหมุน. citeturn6view0turn9view0
- **เกมบุก:** default ที่แข็งแรงคือ **1-3-1** โดยให้ทีมยึด **Garage/Mid** เป็น pivot แล้วค่อยเลือก **C split (Long + Garage)**, **A split (Long + Short/Sewers)** หรือ fast **B punish** ถ้าฝั่งรับว่างกลาง map จุดที่ควรชนะให้ได้คือ **Garage Window, Mid Courtyard, C Long Cubby, A Short/A Lobby** เพราะมันเป็นพื้นที่เปิด fake และ late pivot ทั้งหมด.

- **Meta ตอนนี้:** Haven ยังอยู่ใน active pool และ Riot ปรับบางจุดเรื่อง wall penetration ใน 12.00 เพื่อลดการยิงสแปมที่เสี่ยงต่ำเกินไป เชิงเมต้า map นี้ยังชอบ initiator ที่เปิดพื้นที่ได้เร็ว, controller ที่หมุน smoke ได้ยืดหยุ่น และ sentinel ที่ไม่บังคับให้ยืนติด utility มากเกินไป. citeturn10view0turn14search16

- **3 Pro Tips:** 1) ถ้าคุณมี Garage control เกม C และ B จะง่ายขึ้นทั้งคู่, 2) A retake ที่ดีต้องมาพร้อม Heaven + Short pinch ไม่ใช่ลงทีละคน, 3) การโชว์ spike ทาง C แล้วกลับ A ยังเป็นหนึ่งใน fake ที่คุ้มค่าที่สุดเสมอ

**TL;DR:** Haven ชนะด้วย **ข้อมูล + fake + การหมุนที่ไม่ panic**; ทีมที่รีบ stack เร็วเกินไปจะโดนหลอกง่ายมาก

### Lotus

- **ภาพรวม:** Lotus เป็นแผนที่ 3 ไซต์ที่มีกลไกพิเศษเยอะที่สุด map หนึ่ง: **rotating doors**, **breakable wall**, และ **silent drop** ฝั่ง A; Red Bull มองว่า Lotus ค่อนข้างสมดุล แต่เป็น map ที่ favors เกมเร็วและการกดพื้นที่เป็นกลุ่มมากกว่าการ lurk ยาวเดี่ยวๆ. citeturn5view0turn9view0
- **เกมบุก:** execute ที่นิ่งที่สุดคือการเล่นเป็น **4-1 หรือ 5-man grouped hit** โดยใช้ **A Main + Rubble/Tree**, **C Mound + Bend**, หรือ quick **B crunch** หลังทำลายกำแพงตาม timing ของทีมเอง จุดที่ควรยึดให้ได้คือ **Rubble, Tree, Mound, Bend, A Drop** เพราะพื้นที่เหล่านี้เป็นตัวตัดว่าฝั่งรับจะ retake เชื่อมกันได้หรือไม่.
- **เกมรับ:** setup มาตรฐานคือ 2A / 1B / 2C แบบยืดหยุ่น หรือมีผู้เล่นคนหนึ่งคอยโยกตามเสียงประตู early info ที่ดีมากคือการ contest **A Rubble** หรือ **C Mound** เพราะถ้าเสียสองจุดนี้เร็ว defense จะถูกบีบให้เล่น retake เกือบทั้งรอบ ประตูหมุนช่วย rotate เร็วจริง แต่ก็เปิดช่องให้โดน fake เสียงได้ตลอด.
- **Meta ตอนนี้:** Lotus กลับเข้าคิวใน 12.05 พร้อมการเปลี่ยน **A Site** หลายจุดเพื่อลด wall spam หน้า barrier, ช่วยฝั่งบุกออกจาก Lobby ดีขึ้น และเปลี่ยน plant zone เพื่อไม่ให้ plant ฟรีสำหรับ breakable/link ง่ายเกินไป จึงทำให้การยึด Rubble/Tree มีน้ำหนักมากขึ้นกว่าเดิม. citeturn2view0turn14search16
- **Do / Don’t:** **Do:** เล่น grouped pressure และใช้ประตูเป็นเครื่องมือบีบการตัดสินใจของศัตรู. **Don’t:** lurk นานเกินไปโดยที่ทีมที่เหลือยังไม่สร้างแรงกดอีกฝั่ง.
- **3 Pro Tips:** 1) อย่าทุบกำแพง B แบบนิสัย—ทุบเมื่อมันผูกกับ execute จริง, 2) utility trap บางอย่างจะโดนประตูหมุนทำลาย คิดเรื่องนี้ก่อนเซ็ตพื้นที่, 3) post-plant บน Lotus มักชนะด้วย “connector pinch” ไม่ใช่ถอย main ทั้งหมด

**TL;DR:** Lotus ให้รางวัลกับ **ทีมที่เข้าพร้อมกันและใช้ประตูกับกำแพงเป็น tempo tool** มากกว่าทีมที่เล่น lurk ยาว

### Pearl

- **ภาพรวม:** Pearl เป็นแผนที่ 2 ไซต์ที่ Riot เรียกว่าเป็น map แบบ “ไม่มี mechanics” และ Red Bull ชี้ชัดว่ามันถูกออกแบบมาให้ **mid control** กับ coordinated attack มีความสำคัญสูง เพราะ rotation ไกลและปีกแผนที่ยาว. citeturn5view0turn9view0
- **เกมบุก:** ภาพที่ดีที่สุดคือ default เพื่อคุม **Art / Mid Plaza** ก่อน จากนั้นค่อยแตกเป็น **A split (Art + A Main เข้าหา Dugout/Flowers)** หรือ **B split (Connector + B Long/Halls)** controller ที่ตัดมุมได้ยาวเช่น Viper/Harbor มีมูลค่าสูงมาก จุดที่ฝั่งบุกควรเอาให้ได้คือ **Art, Connector, Halls, Dugout**.
- **เกมรับ:** defense ที่ดีต้องไม่ยก mid ฟรี และต้องมี utility รองรับ B เพราะ **B hold คนเดียวมักอันตราย** ถ้าฝั่งบุกมี utility ครบการยืนลึกหลังไซต์จะโดนไล่ออกง่ายกว่าเล่น retake layered จากหลายมุม แผนที่นี้ลงโทษ defense ที่ “เชื่อเสียงมากกว่า info” เพราะ fake rotation ทำได้แรง.
- **Meta ตอนนี้:** Pearl กลับเข้าสู่ ranked pool ใน 11.08 และ Riot ปรับ **B Site** เพื่อลดความเกินพอดีของการถือ post-plant จาก B Main ระยะลึก ทำให้ on-site fight และ retake มีความแฟร์ขึ้น แปลว่าเกม Pearl ตอนนี้ยังชนะด้วย mid control เหมือนเดิม แต่การถอยไปเล่น long post-plant ฟรีๆ ไม่ปลอดภัยเท่าเดิมแล้ว. citeturn15view0turn14search16
- **Do / Don’t:** **Do:** ชนะ Art ก่อนจะเรียกคอมมิตไซต์. **Don’t:** ปล่อย Halls หรือ Mid free แล้วหวังว่าไฟต์หน้าไซต์จะพอ.
- **3 Pro Tips:** 1) ถ้าคุณ plant A โดยไม่เคลียร์ Dugout ให้ถือว่ารอบยังไม่จบ, 2) B post-plant ที่ดีต้องมีคนคุม Connector ไม่ใช่ค้างแต่ Long, 3) defense ที่แข็งแรงบน Pearl มักเล่น “info first, re-peek later” มากกว่าเตอร์นแรกยิงเอาตาย

**TL;DR:** Pearl คือ map ของ **mid-first macro**; ถ้าไม่คุมกลาง คุณกำลังบุกหรือรับแบบเสียเปรียบทั้งรอบ

### Split

- **ภาพรวม:** Split เป็นแผนที่ 2 ไซต์ที่ขึ้นชื่อเรื่อง verticality และการคุม mid ผ่าน **Mail/Vents/Ropes**; Riot เน้น elevation และ Red Bull มองว่าเป็นแผนที่ที่ค่อนข้างเอียงไปทางฝั่งรับ ดังนั้นฝั่งบุกต้องไม่ข้ามขั้นตอนการชนะ mid เด็ดขาด. citeturn5view0turn8view0
- **เกมบุก:** route ที่เสถียรที่สุดเกือบทุกครั้งคือ **คุม Mid ก่อน** แล้วค่อย split **A ผ่าน Ramp + Main** หรือ **B ผ่าน Heaven + Main** utility ที่คุ้มที่สุดไม่ใช่ utility ลึกสุดในไซต์ แต่เป็น utility ที่พาคุณยึด **Mail, Vents, Rafters/Heaven** ซึ่งเป็นแกน retake/hold ทั้งหมดของแผนที่นี้.
- **เกมรับ:** setup พื้นฐานคือ **2-1-2** โดยมีผู้เล่นกลางหนึ่งคนหรือมี utility mid หนาเป็นพิเศษ Anchor ที่แข็งคือ **A Rafters/Screens** กับ **B Heaven/Back Site**; defense ที่ดีจะไม่ยอมเสีย mid ฟรี และจะ rotate ผ่าน ropes/vents เร็วกว่า map ทั่วไป แปลว่าฝั่งบุกต้องคิดถึง flank timing เสมอ.
- **Meta ตอนนี้:** Split ถูกนำกลับเข้าคิวใน 11.08 และยังอยู่ใน pool ปัจจุบัน หลักเมต้าของมันยังเหมือนเดิม: sentinel ที่ล็อก flank/ไซต์ดี, controller ที่ควบคุมแนวดิ่งได้, และ duelist/initiator ที่พาออกจาก choke แคบได้มีค่ามากที่สุด. citeturn15view0turn14search16
- **Do / Don’t:** **Do:** ใช้รอบต้นเกมเพื่อชนะ Mail/Vents ก่อน. **Don’t:** ชนไซต์จาก main เพียวๆ โดยไม่เปิด heaven/rafters pressure.
- **3 Pro Tips:** 1) Split ไม่ได้ชนะด้วย “เร็ว” แต่ชนะด้วย “แน่นและครบ”, 2) utility เปิด Rafters/Heaven มักคุ้มกว่ายิง utility ลง default plant spot, 3) ถ้าคุณคุม ropes ได้ดี retake ของฝั่งรับจะช้าลงอย่างเห็นได้ชัด

**TL;DR:** Split คือ map ของ **mid discipline**; ทีมที่ยอมแพ้ mid จะต้องบุกผ่าน choke แคบที่สุดในเกมตลอดรอบ

### Bind

- **ภาพรวม:** Bind เป็นแผนที่ 2 ไซต์ที่ **ไม่มี mid** และใช้ **one-way teleporters** เป็นกลไกหลัก ทำให้แมพนี้เน้นการตัดสินใจ “ซ้ายหรือขวา” มากกว่า “กลางหรือข้าง”; Red Bull มองว่าแม้เลย์เอาต์จะดูง่าย แต่สำหรับฝั่งบุกมันเป็น map ที่ควบคุมยากกว่าที่หน้าตาแสดงออกมา. citeturn6view0turn9view0
- **เกมบุก:** default ที่ดีที่สุดคือการขึ้นแรงกด “ทั้งสองซีก” ก่อน ไม่ใช่ commit 5 คนเร็วเกินไป; route หลักคือ **A ผ่าน Showers + Short** หรือ **B ผ่าน Hookah + Long** โดย controller ต้อง smoke **Heaven/Lamps** หรือ **CT/Elbow** ให้เรียบร้อยก่อน duel ตัวแรก จุดที่ต้องยึดคือ **Showers, A Lamps/U-Hall, Hookah, B Long, Elbow**.
- **เกมรับ:** Bind เหมาะกับ setup 2-1-2 หรือ 2A/2B/1 info player ที่พร้อมอ่านเสียง teleport ฝั่งรับหมุนเร็วได้หากอ่านถูก แต่ถ้า overrotate จะโดน fake ได้ทันที Anchor สำคัญคือ **U-Hall** และ **Back B / Elbow**; retake ที่ดีควรเข้าพร้อมกันจากอย่างน้อยสองทิศ ไม่เช่นนั้นจะโดน Clear ทีละมุมง่ายมาก.
- **Meta ตอนนี้:** Riot ถอด Bind ออกจาก Competitive/Deathmatch ใน 12.08 แทนที่ด้วย Ascent แต่ในเชิงโครงสร้างมันยังเป็น map ที่ให้ค่ากับ controller smoke หนา, utility ล้างมุม close และผู้เล่นที่เข้าใจการใช้เสียง teleporter เพื่อเฟกการหมุน. citeturn1view0
- **Do / Don’t:** **Do:** เล่น default/split ให้คนเห็นทั้ง Showers-Short และ Hookah-Long. **Don’t:** ส่ง 5 คนชน choke เดียวโดยไม่บีบอีกฝั่งหรือไม่ทำ fake pressure.
- **3 Pro Tips:** 1) ถ้าจะ plant A ให้เอา Lamps ให้ได้ก่อน—ไม่งั้น post-plant จะเปราะมาก, 2) teleporter noise เป็น “เครื่องมือบังคับ rotate” ไม่ใช่คำสั่งว่าคุณต้อง rotate จริง, 3) B retake ดีที่สุดเมื่อมีคนมา late จาก Long หรือ flank lane เพิ่มอีกชั้น

**TL;DR:** Bind ชนะด้วย **default pressure + teleporter mind game**; ทีมที่ commit เร็วเกินไปจะทำให้ตัวเลือกของตัวเองแคบลงทันที

### Icebox

- **ภาพรวม:** Icebox เป็นแผนที่ 2 ไซต์ที่ขึ้นชื่อเรื่อง verticality, ziplines และ plant site ที่หลายชั้น ทำให้มันเป็นหนึ่งในแผนที่ที่ต้อง “เล่นมุมสูง” ให้เป็นจริงๆ Red Bull มองว่าแม้ผู้เล่นใหม่จะลำบากกับมัน แต่ฝั่งบุกมักรู้สึกได้พื้นที่ทำการบ้านมากกว่า maps ชน choke แคบ. citeturn5view0turn8view0
- **เกมบุก:** default ที่ดีมักมีแรงกด **Tube/Kitchen** ควบคู่กับ lane หลัก แล้วค่อยเลือก **A ผ่าน Belt/Main** หรือ **B ผ่าน Long/Yellow + Kitchen lurk** จุดที่ควรยึดให้ได้คือ **Tube, Kitchen, Belt, Nest, Yellow, Snowman**; แผนที่นี้ reward ผู้เล่นที่อดทนทำ lurk มากพอๆ กับทีมที่ execute พร้อมกัน.
- **เกมรับ:** defense ที่ดีบน Icebox ไม่ควรรอในไซต์ล้วนๆ ต้องมีการ contest พื้นที่อย่าง **Tube, B Long, A Belt** เพื่อไม่ให้ฝั่งบุกตั้ง default สบายเกินไป Anchor ควรเตรียมถอยสู่ชั้นสอง/มุมสูง เพราะ retake เชิงตรงจากพื้นล่างมักเสียเปรียบมาก.
- **Meta ตอนนี้:** Icebox ถูกถอดออกจาก Competitive/Deathmatch ใน 11.04 แต่เชิงเมต้ายังเป็นแผนที่ของ controller แบบ wall, Sage-style plant denial/setup, initiator scan และผู้เล่นที่มั่นใจไฟต์แนวดิ่ง ข้อผิดพลาดหลักคือเล่นเหมือน map แบนราบ—ซึ่งเท่ากับเสีย half the map ไปฟรีๆ. citeturn16view0
- **Do / Don’t:** **Do:** ใช้ lurk และ vertical timing ให้เต็มที่. **Don’t:** ยืนทั้งหมดระดับพื้นเดียวกันหรือปล่อย Kitchen/Tube โดยไม่มีแผนคุม.
- **3 Pro Tips:** 1) Plant ที่ดีต้องสอดคล้องกับพื้นที่ที่คุณ “คุมจริง” เช่น Belt/Yellow/Nest, 2) defense retake ควรใช้การหนีบจากหลายชั้น ไม่ใช่วิ่งจาก spawn ตรงๆ, 3) ถ้าคุณกลัวมุมสูงเกินไป คุณจะเสีย map control เร็วมากใน Icebox

**TL;DR:** Icebox เป็นแผนที่ของ **vertical control + patient lurk**; คนที่เล่นแต่ชั้นล่างจะเสียเปรียบแทบทุก phase

### Sunset

- **ภาพรวม:** Sunset เป็นแผนที่ 2 ไซต์แบบสามเลนที่มีโครงสร้างใกล้ Ascent พอสมควร และมีประตูทำลายได้เป็นกลไกเด่น; Red Bull มองว่ามันเป็น map “traditional” มาก ไม่มี gimmick หนัก แต่มีรายละเอียดเล็กๆ อย่างกล่องบางแบบที่ jump-up ได้สูงกว่าปกติ. citeturn5view0turn9view0
- **เกมบุก:** key ของ Sunset คือ **Mid Courtyard/Boba/Market access** execute ที่สะอาดที่สุดมักเป็น **B split ผ่าน Mid + B Main** หลังได้ mid แล้ว หรือ heavy **A hit** ถ้าศัตรูลงคน B เยอะ จุดที่ควรชนะให้ได้คือ **Mid Courtyard, Boba/Market, Tiles และทางเข้าไซต์ B** เพราะถ้าเสียกลาง map ฝั่งบุกจะเข้าสองไซต์แบบแฟร์ยากมาก.
- **เกมรับ:** sentinel anchor ที่ B มีค่ามาก เพราะ B เป็นไซต์ที่ค้างรับคนเดียวลำบากกว่า A พอสมควร defense ที่ดีควรมีผู้เล่นกด mid หรืออย่างน้อย deny mid info ตั้งแต่ต้นรอบ และใช้ประตู/ทางเชื่อมเพื่อ isolate การ retake เป็นระยะๆ.
- **Meta ตอนนี้:** Sunset ถูกนำออกจาก Competitive/Deathmatch ใน 12.00 เพื่อเปิดทางให้ Breeze กลับเข้ามา แม้ตอนนี้อยู่นอก pool แต่ลักษณะการเล่นยังคงเป็น map mid-control แบบ textbook ซึ่งเหมาะแก่การฝึกพื้นฐาน macro ของทีมมาก. citeturn10view0
- **Do / Don’t:** **Do:** วางแผน mid control ทุกครั้งก่อนเรียก B split. **Don’t:** เดิน B Main เป็นขบวนโดยไม่เปิดแรงกด mid หรือไม่เช็คมุมใกล้.
- **3 Pro Tips:** 1) ถ้าได้ mid Sunset จะ “เปิด” เร็วมาก, 2) ใช้ประตูทำลายได้เพื่อ isolate ฝั่ง retake ไม่ใช่ปิดเพราะเคยชิน, 3) B anchor ที่ดีไม่ต้องยืนตาย แต่ต้องซื้อเวลาให้ mid rotate มาช่วยทัน

**TL;DR:** Sunset คือ **Ascent รุ่นมุมแคบและมุมใกล้เยอะกว่า**; mid control ยังเป็นคำตอบเดิมเกือบทั้งหมด

### Abyss

- **ภาพรวม:** Abyss เป็นแผนที่ 2 ไซต์ที่มี **death drops** และเส้นยิงยาวจำนวนมาก ทำให้การเคลื่อนที่ผิดพลาด “ตายเอง” ได้พอๆ กับโดนยิง แกนของ map คือการคุมพื้นที่เสี่ยงอย่างเป็นระบบ และการเคลียร์มุมสูงก่อนพยายามเข้าไซต์. citeturn5view0turn9view0
- **เกมบุก:** early **mid control** มีค่ามากที่สุด เพราะเปิดการเข้าทั้ง A และ B ได้ พร้อมลดโอกาสโดนยิงยาวฟรีจากมุมสูง execute ที่ดีต้องมี smoke สำหรับ crossing lane เปิดและ info utility นำหน้าก่อนทุก swing จุดสำคัญคือ **Mid, high ground ต่างๆ, long corridors หน้าไซต์** โดยเฉพาะถ้าฝั่งรับถือ Operator.
- **เกมรับ:** Red Bull ชี้ว่าการยืน high ground ยังเป็นทางเลือกที่ดีที่สุดหลายจุด และ **B solo hold ยาก** กว่า A พอสมควร ดังนั้นฝั่งรับควรยอมเล่น retake บางรอบแทนการฝืนยืนลึกคนเดียว early info play ควรทำเมื่อมี utility รองรับเท่านั้น เพราะการผิดจังหวะหนึ่งครั้งอาจเสียทั้งพื้นที่และชีวิตจาก edge danger พร้อมกัน.
- **Meta ตอนนี้:** Riot ปรับ Abyss ใน 11.08 โดยแก้ **B-site on-site play** และปรับ **mid hallway pathing** ให้ทั้งสองฝั่งเล่น macro ได้ลื่นขึ้น ก่อนจะนำ Abyss ออกจาก Competitive/Deathmatch ใน 12.05 เชิงเมต้า map นี้ยังชอบผู้เล่นที่ยิงยาวนิ่ง, initiator ข้อมูลดี และ controller ที่สร้าง safe crossing ได้. citeturn15view0turn2view0
- **Do / Don’t:** **Do:** เคลียร์มุมสูงและ edge-adjacent angles ด้วย utility ก่อนทุกครั้ง. **Don’t:** force duel ระยะไกลแบบ dry หรือยืนเบียดกันเกินไปบนขอบพื้นที่เสี่ยง.
- **3 Pro Tips:** 1) ถ้าคุณไม่แน่ใจว่ามุมนี้มีศัตรูไหม ให้โยน utility ก่อน—Abyss ลงโทษความเดามากกว่า map อื่น, 2) plant บน B ควรผูกกับแผนคุม on-site ไม่ใช่หวัง main post-plant อย่างเดียว, 3) spacing ใกล้ขอบสำคัญมาก เพราะ body-block กันเองอาจทำให้หลุดตกได้

**TL;DR:** Abyss เป็นแผนที่ของ **precision, spacing และ high-ground respect**; ถ้าคุณรีบเกินไป แผนที่จะฆ่าคุณเอง

### Corrode

- **ภาพรวม:** Corrode คือแผนที่มาตรฐานใหม่ที่ Riot เพิ่มใน 11.00 โดยอธิบายว่าเป็นแผนที่ 2 ไซต์ สามเลน มี layered defenses สูงในธีมเมืองปราสาทฝรั่งเศส ส่วน Red Bull มองว่ามันเป็น map ที่ “traditional” กว่าที่หน้าตาใหม่ของมันบอก แต่เน้น **positioning** และการคุมมุมแน่นๆ มากกว่า flashy utility chain. citeturn5view0turn4search0turn9view0
- **เกมบุก:** โครงบุกที่ดีที่สุดคือค่อยๆ ล้างมุมชิดให้ครบก่อน เพราะ map นี้มี blind corners เยอะมาก A มีทางเข้ากว้างกว่า บุกง่ายกว่านิดหนึ่ง ส่วน **B เป็นไซต์ที่ถือยากกว่าแต่ reward สูงกว่า** ถ้าเข้าเร็วจังหวะดี จุดที่ควรยึดคือ **mid pivot, มุมสูงใน B, และ corridor แคบที่เชื่อมเข้าหน้าไซต์**.
- **เกมรับ:** defense ของ Corrode ไม่ควรเล่นโดดเดี่ยวมาก เพราะทุก lane มีมุมเทรดระยะใกล้เยอะ Setup ที่ดีควรมี fallback layer ชัด โดยเฉพาะ B ที่ควรได้รับ resource มากกว่า A ฝั่งรับจะเก่งเมื่อ “ซื้อเวลาและรอเพื่อน” มากกว่า “ยืนรับมุมแรกให้ตาย.”
- **Meta ตอนนี้:** Riot ปรับวัสดุกำแพงบางจุดของ Corrode ใน 12.00 เพื่อลด low-risk wallbang และใน 12.05 ก็ถอด Corrode ออกจากคิว Competitive/Deathmatch ไปก่อน เชิงเมต้ามันยังเป็นแผนที่ที่ initiator สำหรับล้างมุม close และ controller ที่วาง smoke แบบ layered มีค่ามากกว่าปกติ พร้อมกับอาวุธระยะใกล้ที่น่ากลัวกว่า map โปร่งทั่วไป. citeturn10view0turn2view0
- **Do / Don’t:** **Do:** ล้าง cubby ทุกมุมก่อนข้าม choke และเล่นคู่เมื่อเข้า lane แคบ. **Don’t:** push เดี่ยวหรือถือมุมแรกแบบไม่มี fallback plan.
- **3 Pro Tips:** 1) B hit เร็วมีมูลค่าสูงกว่าที่ผู้เล่นใหม่คาด เพราะ defense ตั้งตัวไม่ทันใน corridor แคบ, 2) ถ้าคุณไม่ใช้ initiator ตรวจมุม close จะเสียเลือดฟรีทุก execute, 3) defense ที่ดีบน Corrode คือถอยทีละชั้นแล้วคุม crossfire—not hero hold

**TL;DR:** Corrode เป็นแผนที่ของ **discipline และ corner-clearing**; คนที่เล่นรีบและเชื่อว่ามุมจะว่าง มักโดนลงโทษหนักมาก
