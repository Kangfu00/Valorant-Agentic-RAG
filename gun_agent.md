# คู่มือภาษาไทยฉบับอ่านง่ายสำหรับเอเจนท์และอาวุธใน VALORANT

## บทสรุปผู้บริหาร

ณ วันที่ 4 พฤษภาคม 2026 รายชื่อเอเจนท์บนหน้าทางการของ entity["company","Riot Games","game developer"] สำหรับ VALORANT มี **29 ตัว** และ roster ปัจจุบันรวมเอเจนท์ใหม่อย่าง **Veto** และ **Miks** แล้ว คู่มือนี้จึงยึด “ชื่อแสดงผลทางการ” ของ Riot เป็นหลักสำหรับชื่อเอเจนท์, role, และชื่อสกิล ส่วนคำแนะนำการใช้สกิล/การซื้อปืนเป็น **ข้อเสนอแนะเชิงวิเคราะห์ทั่วไป** ไม่ใช่เมตาบังคับตายตัว เพราะ Riot ปรับสมดุลเอเจนท์และอาวุธอยู่เรื่อย ๆ ในแพตช์ล่าสุดหลายช่วง เช่น 11.08, 11.10, 12.00, 12.05, 12.06 และ 12.08. citeturn11view0turn18search0turn19search0turn24search0turn34search0turn17search1turn20search0turn18search1

ด้านอาวุธ คู่มือนี้เน้น **อาวุธใน Armory/Buy Menu ของโหมดมาตรฐาน** พร้อมเกราะ โดยใช้ราคาปัจจุบันจากหน้าเครดิต/อาวุธที่อัปเดตของชุมชน และตรวจทานกับหน้า patch notes / บทความทางการของ Riot สำหรับอาวุธหรือไอเท็มที่มีการเพิ่มเข้ามาใหม่หรือมีการปรับราคาอย่างชัดเจน เช่น **Bandit (600)**, **Outlaw (2400)**, **Regen Shield (650)**, **Shorty (300)**, **Stinger (1100)** และ **Ares (1600)**. citeturn43search0turn17search1turn26search2turn28search0turn33view0turn31search1turn31search6

ถ้าต้องการฉบับสั้นที่สุดให้จำ 3 เรื่องนี้ไว้ก่อน  
- **Duelist**: เอาไว้เปิดไฟต์และกินพื้นที่ ต้องเล่น “พร้อมเพื่อน” มากกว่าเล่นคนเดียวมั่ว ๆ  
- **Initiator**: เอาไว้เปิดข้อมูล/แฟลช/บังคับมุม ถ้า utility ออกก่อนเพื่อนบุก 1–2 จังหวะจะคุ้มที่สุด  
- **Controller / Sentinel**: คุมจังหวะรอบและคุมพื้นที่ ซื้อปืนให้เข้ากับระยะยิงที่ตัวละครตัวเองยืนจริง ไม่ใช่ยึดแต่ปืนเมตาอย่างเดียว. citeturn7view0turn11view0

## ขอบเขตและหลักการอ่านคู่มือนี้

คู่มือนี้ใช้ชื่อ, role, และฐานข้อมูล kit จากหน้าทางการของ Riot เป็นหลัก และเติมด้วยคำอธิบายเชิงเล่นจริงให้อ่านง่ายขึ้น โดยในส่วนคำแนะนำ “ควรใช้ยังไง / ใช้ตอนไหนดีที่สุด / ซื้ออะไรดี” เป็นการสังเคราะห์จากคำอธิบายสกิลทางการ, แนวคิด economy ขั้นพื้นฐานของ Riot และบทวิเคราะห์เชิงแท็กติกทั่วไปของเกม. citeturn11view0turn7view0turn19search1turn18search0turn37search2turn35search8turn36search4

มีข้อจำกัดเล็กน้อยที่ควรรู้  
- หน้า Agent ของ Riot ที่ดึงผ่าน web snippet บางตัวแสดงชื่อสกิล/คำอธิบายไม่ครบทุกบรรทัด แต่ยังใช้ยืนยัน role และแกนของ kit ได้  
- **Yoru** มีความไม่สม่ำเสมอบางจุดในพื้นผิวสื่อของ Riot ระหว่างคำเรียก decoy utility แบบ “Fakeout” กับ “Clone”; ในคู่มือนี้จะใช้ **Clone / Fakeout** เพื่อให้ตรงกับสิ่งที่ผู้เล่นคุ้นเคยและสิ่งที่ Riot ใช้ในผิวสื่อช่วงหลัง  
- ตารางอาวุธนี้ **ไม่รวมอาวุธเฉพาะโหมด** อย่าง Golden Gun / Snowball Launcher และจะโฟกัสที่ buy menu ปกติเป็นหลัก. citeturn46search1turn18search1turn40search0

## คู่มือเอเจนท์รายตัว

### Duelists

Duelist ใน roster ปัจจุบันคือ Jett, Phoenix, Raze, Reyna, Neon, Yoru, Iso และ Waylay โดยแกนร่วมกันคือ “สร้างพื้นที่, เปิดไฟต์แรก, บังคับ crosshair ฝ่ายตรงข้ามให้เสียรูป” ดังนั้นสกิลพวกนี้จะคุ้มที่สุดเมื่อใช้ **ก่อนเพื่อนสวิง 0.5–1.5 วินาที** ไม่ใช่ใช้แล้วนานเกินจนทีมตามไม่ทัน. citeturn48search0turn49search6turn48search4turn48search1turn49search8turn49search0turn49search1

#### Jett
- **ชื่อทางการ:** Jett  
- **สกิล**
  - **Cloudburst** — ควันก้อนเล็กออกไวและบิดทิศได้  
    **ใช้ยังไง:** ปิดมุมแรกตอน entry, ปิดมุม trade, หรือคุม half-defuse  
    **ใช้ดีที่สุด:** ตอน dash เข้าพื้นที่, retake, หรือช่วยถอนตัวหลังยิงนัดแรก
  - **Updraft** — ยกตัวขึ้นสูงทันที  
    **ใช้ยังไง:** เปลี่ยนมุมยิงแนวตั้ง, หนี utility ที่พื้น, เล่นคู่กับ knives  
    **ใช้ดีที่สุด:** ตอนจะ surprise angle หรือแก้เสียจังหวะจากโดนบีบมุม
  - **Tailwind** — dash ตามทิศการเคลื่อนที่  
    **ใช้ยังไง:** ยิงเปิด 1 นัดแล้วถอย, dash เข้า site พร้อม smoke, ข้าม choke สั้น ๆ  
    **ใช้ดีที่สุด:** first contact, OP pick, หรือจังหวะที่ทีมพร้อม trade
  - **Blade Storm** — มีดอัลติแม่นและคล่องตัว  
    **ใช้ยังไง:** ใช้แทนปืน full-buy ในรอบ eco/half-buy หรือเล่นบนที่สูง/กลางอากาศ  
    **ใช้ดีที่สุด:** รอบเงินน้อย, รอบต้องการ mobility สูง, หรือเจอศัตรูไม่เต็มเกราะ
- **วิธีเล่น:** เล่นเป็นหัวหอกแต่ห้ามเข้าแบบโดดเดี่ยว ขอ flash / scan ก่อนเสมอ, ใช้ dash เป็น “รีเซ็ตความเสี่ยง” ไม่ใช่รีบกดทุกไฟต์, ถ้าเล่น OP ให้ยึดหลักยิงแล้วเปลี่ยนตำแหน่งทันที
- **ซื้อแนะนำ:** **Eco:** Sheriff หรือ Classic + util / บางแมพใช้ Shorty ได้, **Force/Half:** Marshal หรือ Spectre, **Full:** Vandal หรือ Operator, **เกราะ:** Heavy เป็นหลัก; Regen น่าสนใจถ้าจะเล่น knives / hit-and-run. citeturn48search0turn48search2
